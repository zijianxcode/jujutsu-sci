# 发包归档

这个目录保存已经对外发布或可回滚使用的完整站点发包 zip。

## 归档规则

- `packages/` 只放不可变的版本包，文件名使用 `jujutsu-sci-vX.Y.Z.zip`。
- 每个 zip 必须在本文件登记版本、用途、大小和 SHA-256，方便校验与回滚。
- 如果重新打包同一个版本，不要覆盖旧文件；应提升版本号并补充更新说明。
- 当前 zip 是完整静态站点包，不是 CloudBase 专用包。CloudBase 发布仍以仓库静态文件同步到 `personal-homepage/academy/` 后再部署为准。
